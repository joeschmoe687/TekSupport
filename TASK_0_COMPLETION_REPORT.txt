================================================================================
                                  PROBLEM
================================================================================

Payment system completely blocked by Stripe theme error:

  Error: Your theme isn't set to use Theme.AppCompat or Theme.MaterialComponents

- Appeared when presentPaymentSheet() was called
- Blocked all payment flows: Phone, Video, Text Chat
- Stripe initialized successfully at startup, but failed during payment
- Google Pay availability check failed

================================================================================
                               ROOT CAUSE
================================================================================

Night theme file used incompatible Android theme:

  File: android/app/src/main/res/values-night/styles.xml
  
  ❌ BEFORE: parent="@android:style/Theme.Black.NoTitleBar"
  ✅ AFTER:  parent="Theme.AppCompat.Light.NoActionBar"

When devices run in dark mode, Android uses the night theme. Stripe requires
AppCompat themes for Google Pay integration. The night theme override caused
the compatibility check to fail.

================================================================================
                               SOLUTION
================================================================================

Three-part fix with multiple safety measures:

1. FIXED NIGHT THEME (2 lines changed)
   - File: android/app/src/main/res/values-night/styles.xml
   - Changed LaunchTheme parent to Theme.AppCompat.Light.NoActionBar
   - Changed NormalTheme parent to Theme.AppCompat.Light.NoActionBar
   
2. ADDED PROGUARD RULES (6 lines added)
   - File: android/app/proguard-rules.pro
   - Preserves AppCompat classes during minification
   - Prevents release builds from stripping theme classes
   
3. ADDED THEME PREFERENCE (5 lines added)
   - File: android/app/src/main/AndroidManifest.xml
   - Added prefers_colorscheme="light" metadata
   - Safety measure to prevent future theme conflicts
