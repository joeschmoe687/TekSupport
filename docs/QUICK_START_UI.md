# Quick Start Guide - New UI Features

## 🌞 Light Theme (for Outdoor Work)

### When to Use
- Working on rooftop units in bright sunlight
- Reading gauges on outdoor condensers
- Any bright outdoor environment

### How to Enable
1. Open any screen with theme support
2. Look for the sun/moon icon in the app bar
3. Tap to toggle between light and dark modes
4. Your choice is saved automatically

### Benefits
- **High Contrast**: Dark text on light background
- **Easy Reading**: No squinting in bright light
- **Professional Look**: Clean, modern appearance

---

## 🎯 Analog Gauge Mode

### When to Use
- Prefer traditional dial gauges
- Want quick visual feedback
- Like to see pressure "movement"

### How to Enable
1. Open the **Gauges** screen
2. Tap the gauge/speedometer icon in the app bar
3. Gauges switch from digital boxes to round dials
4. Tap again to switch back to digital

### What You See
- **Round Dials**: Classic analog appearance
- **Colored Needles**: Red needle shows current reading
- **Tick Marks**: Easy to read scale
- **Same Data**: All sensor info still visible

### Default
Digital mode is the default. Your preference is saved.

---

## ⚖️ Scale Auto-Display

### What It Does
When you connect a Bluetooth scale while on the Gauges screen, the scale weight automatically appears at the bottom. No need to switch screens!

### Features
- **Auto-Appears**: Pops up when scale connects
- **Live Updates**: Weight updates in real-time
- **Connection Status**: Shows signal strength
- **Battery Level**: Scale battery percentage
- **Smart Memory**: Remembers last weight when disconnected

### How It Works
1. Open the **Gauges** screen
2. Turn on your Bluetooth scale
3. Scale widget appears automatically at bottom
4. See both gauges AND scale weight together

### Disconnection Behavior
- If you walk away (out of range), scale shows last known weight
- Widget turns gray to indicate disconnection
- Walk back in range - automatically reconnects
- Weight picks up right where it left off

### Why This Helps
- **One Screen**: See everything at once
- **No Switching**: Stay on gauges while charging
- **Work Freely**: Move around without worry
- **Quick Glance**: Check scale without tapping

## 💡 Tips & Tricks

### Best Practices

**Light Theme:**
- Switch to light mode before going outside
- Switch back to dark mode when you come inside
- Saves battery in dark mode (OLED screens)

**Analog Gauges:**
- Use analog for quick pressure checks
- Use digital for precise readings
- Great for showing customers

**Scale Display:**
- Keep gauges screen open while charging
- Scale automatically reconnects when back in range
- Last weight shown when temporarily disconnected

### Common Questions

**Q: Does theme affect battery life?**
A: Dark mode uses slightly less battery on OLED screens.

**Q: Can I use analog gauges with light theme?**
A: Yes! All features work together in any combination.

**Q: What if my scale won't show up?**
A: Make sure Bluetooth is on and scale is powered up. It detects automatically.

**Q: Do I need to manually connect the scale?**
A: No! It connects automatically when in range of the gauge screen.

**Q: Will my gauge mode setting save?**
A: Yes, your preference for digital/analog saves automatically.

## 🔧 Troubleshooting

### Light Theme Too Bright at Night
**Solution:** Switch back to dark mode. The theme toggle is always available.

### Analog Gauges Not Showing
**Solution:** Tap the gauge icon in the app bar. Make sure you're on the Gauges screen.

### Scale Not Appearing
**Check:**
- Scale is powered on
- Bluetooth is enabled
- You're on the Gauges screen (not Scale screen)
- Scale is within Bluetooth range (~30 feet)

### Scale Shows Gray "Last Known Weight"
**This is normal!** It means:
- Scale is out of range OR
- Scale powered off
- Walk back in range to reconnect
- Last weight is preserved

## 🎬 Typical Workflow

### Charging Refrigerant (Outdoor)

1. Walk outside to condenser
2. **Switch to light theme** (sun icon)
3. Open **Gauges** screen
4. Connect pressure probes
5. Turn on **Bluetooth scale**
   - Scale widget auto-appears at bottom
6. Start charging refrigerant
7. Monitor gauges and scale weight **on same screen**
8. Walk inside to check indoor temps
   - Scale shows last weight (grayed)
9. Walk back outside
   - Scale reconnects automatically
10. Continue charging
11. When done, scale disconnects when powered off

### Using Analog Gauges

1. Open **Gauges** screen
2. Tap **gauge icon** (top right)
3. Watch analog dials
4. Show customer the familiar dial interface
5. Tap icon again to return to digital
